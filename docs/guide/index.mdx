# YK-TABLE

## YK-TABLE

基于 Element-UI Table 组件封装的表格组件，可无缝支持 element 的 table 组件。

这个组件主要是为了将每次写table时那些重复的代码放在组件内部，使用时通过配置的方式去定制table。

### 基本使用

首先是完全支持与 el-table 相同的使用方式的，就是说如果你直接把原来写的 <el-table></el-table> 只改个标签名，改为 <yk-table></yk-table>，也是能正常渲染的，传参、事件都完全兼容。

#### 使用姿势

```js
<template>
  <yk-table
    border
    :data="tableData"
    :columns="columns"
  ></yk-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [
        {id: 1, name: "Niko"},
        {id: 2, name: "Nikola"},
        {id: 3, name: "Nikolala"}
      ],
      columns: [
        {label: "编号", value: "id"},
        {label: "姓名", value: "name"}
      ]
    }
  },
}
</script>
```
